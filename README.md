## 文明详细内容

### 璃月
#### 领袖
- 岩王帝君-**尘世闲游**-摩拉克斯
    - **特色能力**
        **匪石弗转**:"韬玉之石,可明八荒;灿若天星,纵横无双."摩拉克斯治下城市中每个回合每有一个区域未遭到破坏,则根据未被破坏区域的数量获得金币,科技值,文化值和生产力.
    - **议程**
        **坚名约束**:喜欢信守盟约的国家,讨厌背弃盟约的国家.
    - **宗教**
        
    
#### 总督
- 七星玉衡-**霆霓快雨**-刻晴
    - 替代单位:测量师
    - 头衔:
        - **雷厉**:在城市中修建区域时+30%生产力.
        - **苛捐**:此城市中生产的所有建造者将+1建造次数.
        - **登楼**:此城市的每个社区与特色区域+2住房。此城市的每个运河与堤坝+1宜居度.
        - **调律**:每个未改良的地貌可向该城市提供+5金币.该城市中与未改良地貌相邻的单元格可获得+2魅力.
        - **廉贞**:环境效应无法对此城市的改良设施、建筑与区域造成伤害.
- 七星天权-**掩月天权**-凝光
    - 替代单位:金融家
    - 头衔
        - **天权崩玉**:城市中每位公民每回合可带来3金币的收入.
        - **星移势转**:城市中商业中心和港口的相邻加成加倍.
        - **奇石珠宝录**:城市中每有一个奢侈资源,则每回合增加5金币收入.
        - **千金掷**:城市获得新单元格的速度加快.经过城市的每条外国贸易路线每回合产出的金币+3.
        - **储之千日,用之一刻**:城市可使用金币购买区域.
- 月海总秘-**循循守月**-甘雨
    - 替代单位:总务官
    - 描述:
        - **饮露**:城市中格位收获和移除地貌的产出+50%.
        - **获麟**:城市中生产的开拓者不计入人口.
        - **云行**:城市的增长+20%.以此处为终点的贸易路线（属于玩家）为起源城市+2食物.
        - **西狩**:此城市可从附近所有拥有区域加成的工业区处获得生产力，而不仅是从首个建成的工业区处获得.
        - **折草**:生产单位所需战略资源花费减少80%.

#### 特殊单位
- **千岩军**
    - 替代单位:枪兵
    - 特色能力
        **同袍**:「千岩牢固，重嶂不移，干城戎甲，靖妖闲邪.」编队里每多一人,则攻击力提高3.
- **夜叉**
    - 替代单位:
    - 特色能力
        **杀生护法**:「夜叉者，璃月之仙兽，性凶怖悍战，杀生而护法.然诸夜叉虽有大威能，亦毋免为业障困，为遗恚染.」降低10自身生命值,提高20攻击力.

#### 奇观
- 群玉阁-贝伦塔
    - 描述:七星之天权凝光所造的空中楼阁."终有一天,我会让群玉阁的影子,笼罩七国."
    - 特色能力
        **高阁入云坐看星**:每回合金币收入+5,相邻五个单元格内城市贸易路线+2.
- 孤云阁-兵马俑相似
    - 描述:岩王帝君封印旋涡之魔神奥赛尔的地方.
    - 特色能力
        **岩枪**:目前所有单位获得1级强化升级,军队单位攻击力+5.
- 层岩巨渊-博尔戈尔山
    - 描述:
    - 特色能力
        **藏金**:每回合奖励6点铁.为6个单元格内的所有城市提供+4信仰值.必须建在山脉上.
